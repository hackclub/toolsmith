What's this about?

Welcome to Toolsmith! 🧰

Toolsmith is a limited You Ship, We Ship program where you build an awesome MCP tool and get $10 in AI credits for platforms like OpenAI, Anthropic, and more!

So how does this work?

Great question! It's actually really simple:

🚀 You Ship

- MCP server with at least one tool.

- Spend at least 2 hrs.

- Publish it to the gallary.

🎁 We Ship

- $10 of AI Credits for OpenAI, Anthropic, Gemini, etc.

Sounds cool! So, what are MCP tools?

MCP (Model Context Protocol) tools are like superpowers for AI assistants! 🦸‍♂️

Think of them as plugins that give AI new abilities. Instead of just chatting, AI can now:

🌤️

Weather tool

Check the weather anywhere.

📁

File manager

Organize & search files on your own computer.

💬

Slack integration

Allow the AI to send messages as you.
